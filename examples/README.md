# Examples

This directory is reserved for safe, intentionally shared example inputs and outputs.

Do not commit private/raw lifting videos by default. Squat videos may reveal identity, location, gym membership, health status, or other sensitive information.

Recommended practice:

- Keep raw personal videos outside the repository.
- Use short, consented, non-sensitive clips for demos.
- Prefer synthetic, public-domain, or explicitly licensed sample media when possible.
- Store generated annotated outputs only when they are safe to share.

The initial detector should work with videos uploaded into Colab at runtime, so this directory can remain empty until safe examples are selected.

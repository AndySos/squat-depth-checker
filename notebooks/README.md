# Notebooks

The first Colab notebook will live here.

Current notebook:

- `squat_depth_mvp.ipynb`

The notebook is a friendly wrapper around the reusable logic in `src/`. It supports uploading a video in Colab, running pose estimation, finding the bottom frame, classifying squat depth, and displaying annotated output.

When opened from a private GitHub repo, Colab may load only the notebook. In that case, clone or upload the repository files so `src/` is available from the notebook runtime.
